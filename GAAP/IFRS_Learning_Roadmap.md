# IFRS/IAS Deep Learning Roadmap
## Built around Grifols S.A. (FY2025 Consolidated Accounts)

---

## How This Works

Each module covers one IFRS or IAS standard. For every standard we will:

1. **Understand the WHY** — what problem the standard solves and why it exists
2. **Learn the RULES** — recognition, measurement, presentation, disclosure
3. **See it in GRIFOLS** — map the theory to actual line items, notes, and judgments in the 2025 accounts
4. **Flag the LBO angle** — what a financial modeler must watch for when using these numbers

---

## Module Sequence

The order is deliberate. We start with how financial statements are structured, then build up
from individual assets and liabilities to revenue, taxes, financial instruments, leases,
and finally consolidation. Each module assumes you've absorbed the ones before it.
